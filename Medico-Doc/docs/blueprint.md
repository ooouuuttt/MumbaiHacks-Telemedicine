# **App Name**: ArogyaDoc

## Core Features:

- Secure Doctor Authentication: Implement secure doctor login with profile management including specialization, consultation timings, and availability, along with verification of medical licenses.
- Teleconsultation Interface: Enable video/audio consultation with patients and chat support within the browser. Implement appointment scheduling including accept/reject/reschedule functionality and a queue for managing patients.
- Prescription Generator: Create a tool that generates digital prescriptions with e-signature.
- Consultation Dashboard: Show a dashboard displaying number of consultations, and patient demographics.
- Notifications System: Provide notifications for new consultation requests.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a calming, trustworthy feel.
- Background color: Light gray (#F0F0F0) for a clean, professional look.
- Accent color: Teal (#008080) for call-to-action buttons and highlights.
- Body and headline font: 'PT Sans' sans-serif for a modern yet friendly interface.
- Use a set of clean, simple icons that are easily recognizable to represent different functionalities and data points.
- Design a clean, intuitive layout that prioritizes key information and is easy to navigate, even on smaller screens.
- Implement subtle animations to provide feedback to user interactions.
